# 🎯 Примеры использования MuscleIcon

## Базовое использование

```tsx
import { MuscleIcon } from './components/MuscleIcon';

function ExerciseCard() {
  return (
    <div className="flex items-center gap-2">
      <MuscleIcon muscle="biceps" />
      <span>Подъём на бицепс</span>
    </div>
  );
}
```

## С разными размерами

```tsx
// Маленький (16px - размер текста)
<MuscleIcon muscle="chest" size="sm" />

// Средний (20px - чуть крупнее) - ПО УМОЛЧАНИЮ
<MuscleIcon muscle="legs" size="md" />

// Большой (24px)
<MuscleIcon muscle="abs" size="lg" />
```

## Список упражнений

```tsx
function ExerciseList() {
  const exercises = [
    { name: 'Жим лёжа', muscle: 'chest' },
    { name: 'Подтягивания', muscle: 'back' },
    { name: 'Приседания', muscle: 'legs' },
    { name: 'Жим стоя', muscle: 'shoulders' },
  ];

  return (
    <ul className="space-y-2">
      {exercises.map((ex) => (
        <li key={ex.name} className="flex items-center gap-2">
          <MuscleIcon muscle={ex.muscle as any} size="md" />
          <span>{ex.name}</span>
        </li>
      ))}
    </ul>
  );
}
```

## В карточке упражнения

```tsx
function ExerciseCard({ exercise }) {
  return (
    <div className="bg-white rounded-lg shadow p-4">
      <div className="flex items-center justify-between mb-3">
        <h3 className="font-bold text-lg">{exercise.name}</h3>
        <MuscleIcon muscle={exercise.targetMuscle} size="lg" />
      </div>
      
      <div className="text-sm text-gray-600">
        <MuscleIcon muscle={exercise.targetMuscle} size="sm" className="mr-1" />
        {MUSCLE_NAMES[exercise.targetMuscle]}
      </div>
    </div>
  );
}
```

## Выбор мышечной группы

```tsx
import { MUSCLE_EMOJIS, MUSCLE_NAMES } from './constants/muscleEmojis';

function MuscleSelector({ onSelect }) {
  const mainMuscles = ['chest', 'back', 'shoulders', 'arms', 'legs', 'abs'];

  return (
    <div className="grid grid-cols-3 gap-2">
      {mainMuscles.map((muscle) => (
        <button
          key={muscle}
          onClick={() => onSelect(muscle)}
          className="flex flex-col items-center gap-1 p-3 border rounded-lg hover:bg-gray-50"
        >
          <span className="text-3xl">{MUSCLE_EMOJIS[muscle]}</span>
          <span className="text-xs">{MUSCLE_NAMES[muscle]}</span>
        </button>
      ))}
    </div>
  );
}
```

## В форме добавления упражнения

```tsx
function AddExerciseForm() {
  const [selectedMuscle, setSelectedMuscle] = useState('chest');

  return (
    <form className="space-y-4">
      <input 
        type="text" 
        placeholder="Название упражнения"
        className="w-full p-2 border rounded"
      />

      <div>
        <label className="block text-sm font-medium mb-2">
          Целевая мышца
        </label>
        <select 
          value={selectedMuscle}
          onChange={(e) => setSelectedMuscle(e.target.value)}
          className="w-full p-2 border rounded"
        >
          {Object.entries(MUSCLE_NAMES).map(([key, name]) => (
            <option key={key} value={key}>
              {MUSCLE_EMOJIS[key]} {name}
            </option>
          ))}
        </select>
      </div>

      <button className="w-full bg-blue-500 text-white p-2 rounded">
        Добавить
      </button>
    </form>
  );
}
```

## Готово! 🎉

Просто импортируй компонент и используй рядом с упражнениями:

```tsx
<MuscleIcon muscle="biceps" /> Подъём на бицепс
<MuscleIcon muscle="chest" /> Жим лёжа
<MuscleIcon muscle="legs" /> Приседания
```

Размер эмодзи будет как обычный текст - идеально для списков упражнений!
