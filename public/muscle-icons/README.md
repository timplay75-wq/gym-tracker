# 🎯 Иконки мышечных групп

## Где скачать бесплатные PNG иконки:

### 🔥 Рекомендуется: Icons8
1. Открой: https://icons8.com/icons/set/muscle
2. Выбери нужные иконки (бицепс, грудные, ноги и т.д.)
3. Нажми "Download" → выбери размер **48px** или **64px**
4. Скачай в PNG формате (бесплатно с attribution)

### 📦 Альтернатива: Flaticon
1. Открой: https://www.flaticon.com/free-icons/muscle
2. Фильтр: **Free** + **PNG**
3. Скачай нужные иконки (требуется бесплатная регистрация)
4. Размер: 64px или 128px

### 🎨 Ещё вариант: PNG Repo
1. Открой: https://www.pngrepo.com/
2. Поиск: "muscle icon" или "fitness body"
3. Скачай PNG (все бесплатные, CC0)

## ⚙️ Рекомендуемые параметры:

- **Формат**: PNG (с прозрачным фоном)
- **Размер**: 48-64px (для использования как иконки)
- **Стиль**: Outline (линейный) или Flat (плоский)
- **Цвет**: Чёрный или одноцветный (можно перекрасить через CSS)
