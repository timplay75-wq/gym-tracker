# 📦 Инструкция по добавлению иконок мышц

## Шаг 1: Скачай иконки

Выбери один из сайтов:

### 🔥 Icons8 (Рекомендуется)
```
1. Открой: https://icons8.com/icons/set/muscle
2. Найди иконки: biceps, chest, legs, abs и т.д.
3. Нажми на иконку → Download
4. Размер: 48px или 64px
5. Формат: PNG
6. Скачай бесплатно (понадобится attribution в About странице)
```

### 📦 Flaticon
```
1. Открой: https://www.flaticon.com/search?word=muscle
2. Фильтр: Free icons only
3. Скачай нужные иконки (нужна регистрация)
4. Размер: 64px или 128px, PNG
```

### 🎨 PNG Repo (полностью бесплатные)
```
1. Открой: https://www.pngrepo.com/
2. Поиск: "muscle" / "body parts" / "fitness"
3. Все иконки бесплатные (CC0 лицензия)
4. Скачай PNG файлы
```

---

## Шаг 3: Положи в папку

Скопируй все PNG файлы в папку:
```
public/muscle-icons/
```

Должно получиться так:
```
gym-tracker/
  └── public/
      └── muscle-icons/
          ├── biceps.png
          ├── triceps.png
          ├── chest.png
          ├── back.png
          ├── shoulders.png
          ├── abs.png
          ├── legs.png
          └── ...
```

---

## Шаг 4: Готово! 🎉

Теперь используй в коде:

```tsx
import { MuscleIcon } from './components/MuscleIcon';

// Маленькая иконка (16px)
<MuscleIcon muscle="biceps" size="sm" />

// Средняя (20px) - по умолчанию
<MuscleIcon muscle="chest" />

// Большая (24px)
<MuscleIcon muscle="legs" size="lg" />
```

## 💡 Совет:

Скачай иконки в одном стиле (все outline или все flat), чтобы выглядело единообразно!

Размер 48px или 64px идеально подходит для использования как маленькие иконки рядом с текстом.
