## Шаг 1: Скачать иконки

### 🌟 **Вариант 1: Noun Project (Лучший вариант! ✅)**

**Почему лучший:**
- ✅ Реально работает!
- ✅ Огромный выбор анатомических иконок
- ✅ Проверенная трапеция: https://thenounproject.com/icon/trapezius-2529053/

**Ссылки для скачивания:**

1. **Главный поиск по мышцам:**
   - 🔗 https://thenounproject.com/browse/icons/term/muscle/
   - 🔗 https://thenounproject.com/browse/icons/term/anatomy/
   - 🔗 https://thenounproject.com/browse/icons/term/bodybuilding/

2. **Отдельные мышцы:**
   - Трапеция: https://thenounproject.com/search/icons/?q=trapezius
   - Широчайшие: https://thenounproject.com/search/icons/?q=latissimus
   - Лопатки: https://thenounproject.com/search/icons/?q=shoulder+blade
   - Дельты: https://thenounproject.com/search/icons/?q=deltoid
   - Грудь: https://thenounproject.com/search/icons/?q=pectoral
   - Бицепс: https://thenounproject.com/search/icons/?q=biceps
   - Квадрицепсы: https://thenounproject.com/search/icons/?q=quadriceps

**Как скачать с Noun Project:**
```
1. Открой ссылку (например, trapezius)
2. Кликни на понравившуюся иконку
3. Справа нажми кнопку "Download"
4. Выбери:
   - Free (PNG) - бесплатно, нужна атрибуция
   - Pro (SVG) - $3.99/мес, без атрибуции
5. Для бесплатных: скопируй имя автора
6. Скачай файл
```

**Бесплатно или платно?**
- ✅ Бесплатно: PNG формат, нужна атрибуция в футере
- 💰 Pro ($3.99/мес): SVG формат, без ограничений, без атрибуции
- 🎁 SVG можно конвертировать из PNG онлайн бесплатно!

### 🖱️ **Как создать папки:**

**Windows (PowerShell в VS Code):**
```powershell
# Создать все папки одной командой
New-Item -ItemType Directory -Path "public\icons\muscles\back" -Force
New-Item -ItemType Directory -Path "public\icons\muscles\chest" -Force
New-Item -ItemType Directory -Path "public\icons\muscles\shoulders" -Force
New-Item -ItemType Directory -Path "public\icons\muscles\arms" -Force
New-Item -ItemType Directory -Path "public\icons\muscles\legs" -Force
New-Item -ItemType Directory -Path "public\icons\muscles\core" -Force
```

**Или вручную:**
1. Открой папку `gym-tracker/public/icons/`
2. Создай папку `muscles`
3. Внутри создай: `back`, `chest`, `shoulders`, `arms`, `legs`, `core`

## Шаг 5: Использовать в коде!

### 💻 **Пример 1: Одна мышца**
```tsx
import { ExerciseIcon } from '@/components';

<ExerciseIcon 
  anatomyIcon={{
    category: 'back',
    muscle: 'trapeziusUpper'
  }}
  size="lg"
/>
```

### 💻 **Пример 2: Несколько мышц (жим лёжа)**
```tsx
import { MuscleGroup } from '@/components';
import { EXERCISE_MUSCLE_GROUPS } from '@/constants/muscleIcons';

<MuscleGroup 
  muscles={EXERCISE_MUSCLE_GROUPS.benchPress}
  showLabels
  layout="vertical"
/>
```

### 💻 **Пример 3: Кастомная комбинация**
```tsx
<MuscleGroup 
  muscles={[
    { category: 'back', muscle: 'trapeziusUpper', name: 'Трапеция', isPrimary: true },
    { category: 'back', muscle: 'rhomboids', name: 'Лопатки', isPrimary: true },
  ]}
  size="xl"
  showLabels
/>
```

## 🎨 Оптимизация SVG (опционально)

Если иконки тяжелые (>50KB), оптимизируй их:

### **Онлайн:**
1. Иди на https://jakearchibald.github.io/svgomg/
2. Загрузи SVG файл
3. Включи опции:
   - ✅ Remove viewBox
   - ✅ Remove metadata
   - ✅ Minify styles
4. Скачай оптимизированный файл

### **Командная строка:**
```bash
npm install -g svgo
svgo public/icons/muscles/**/*.svg
```

## 🆘 Частые проблемы

### ❌ **Иконка не отображается**
**Решение:**
1. Проверь путь: `public/icons/muscles/back/trapezius-upper.svg`
2. Проверь имя файла (точное совпадение!)
3. Перезагрузи страницу (Ctrl+F5)

### ❌ **Белый фон вместо прозрачного**
**Решение:**
- Скачай SVG вместо PNG
- Или используй SVG с `transparent background`

### ❌ **Иконки слишком большие**
**Решение:**
- Оптимизируй через SVGOMG
- Или используй меньше иконок за раз
