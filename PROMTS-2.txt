Создай экран выполнения тренировки в реальном времени, который соответствует твоему текущему дизайну приложения:

**Особенности дизайна (с учетом твоих скриншотов):**
- Простой чистый интерфейс с крупными заголовками
- Карточки с тенями (как в твоем приложении)
- Нижняя навигационная панель с 4 табами
- Минималистичный стиль с акцентами на функциональность

**Структура экрана активной тренировки:**

1. **Header**
   - Крупный заголовок "Активная тренировка"
   - Название текущей тренировки (например: "Грудь и трицепс")
   - Таймер тренировки в формате MM:SS
   - Кнопка паузы ⏸️ и завершения ⏹️ справа

2. **Current Exercise Section**
   - Карточка с текущим упражнением:
     - Крупное название упражнения (например: "Жим штанги лежа")
     - Прогресс: "Упражнение 2 из 5"
     - Индикатор подходов: "Подход 1 из 4"

3. **Weight & Reps Input**
   - Простая форма для ввода:
     ```
     Вес: [ 60 ] кг
     Повторения: [ 10 ]
     ```
   - Кнопки "+" и "-" для быстрой регулировки
   - История предыдущих подходов (если есть)

4. **Set Actions**
   - Большая кнопка "Завершить подход" с эмодзи ✅
   - Кнопка "Пропустить" с эмодзи ⏭️

5. **Rest Timer (опционально)**
   - После завершения подхода появляется таймер отдыха:
     ```
     Отдых: 90 сек
     [ Продолжить ]
     ```
   - Обратный отсчет с визуальной индикацией

6. **Exercise List**
   - Список всех упражнений в тренировке
   - Текущее упражнение выделено
   - Завершенные упражнения отмечены галочкой
   - Можно тапнуть для перехода к другому упражнению

7. **Bottom Navigation**
   - Внизу сохраняется та же навигация:
     ```
     Главная | Тренировки | Добавить | Профиль
     ```
   - Текущий экран активной тренировки может быть модальным или отдельным экраном

8. **Completion Screen**
   - При завершении тренировки появляется экран с результатами:
     - Поздравление 🎉
     - Итоговая статистика в карточках:
       ```
       Длительность: 45 мин
       Упражнений: 5
       Подходов: 20
       Тонаж: 2,500 кг
       ```
     - Кнопка "Сохранить и выйти"

**Дополнительно:**
- Сохранение прогресса при переключении экранов
- Простая валидация ввода (вес > 0, повторения > 0)
- Автоматический расчет тоннажа (вес × повторения)
- Минимальные анимации для улучшения UX

Стилизуй под существующий дизайн с использованием:
- Крупных заголовков (#)
- Карточек с тенями
- Простых форм ввода
