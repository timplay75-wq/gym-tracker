### Шаг 3: Проверяй критерии завершения

После каждого промта:

```markdown
## Критерии завершения

- [ ] Код скомпилирован без ошибок TypeScript
- [ ] Работает в light и dark mode
- [ ] Респонсивность на мобильных устройствах
- [ ] Accessibility (ARIA атрибуты, keyboard navigation)
- [ ] Примеры использования протестированы
- [ ] Документация обновлена
```

**Если хотя бы один пункт не выполнен** — промт не завершён, доработай.

Улучшенные промты — это **template**, не догма.

**Можно менять:**
