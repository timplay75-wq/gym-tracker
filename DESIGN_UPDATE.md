# Обновление дизайна - Минималистичный iOS стиль

## 📱 Обзор изменений

Приложение полностью переработано в соответствии с минималистичным iOS дизайном с акцентом на:
- **Чистый, плоский дизайн** (Flat UI 2.0)
- **Большие заголовки** для удобной навигации
- **Скругленные углы** на всех элементах
- **Минимальные тени** вместо объемных эффектов
- **Адаптивность** для мобильных устройств, планшетов и десктопов

## 🎨 Цветовая схема

### Light Mode
- **Фон:** `#FFFFFF` (чистый белый)
- **Вторичный фон:** `#F1F3F2` (очень светло-серый)
- **Основной текст:** `#202328` (темно-серый)
- **Акцентный цвет:** `#202328` (для кнопок и активных элементов)

### Dark Mode
- **Фон:** `#0F172A` (темно-синий серый)
- **Вторичный фон:** `#1E293B`
- **Основной текст:** `#F8FAFC` (почти белый)
- **Акцентный цвет:** `#202328`

## 🎯 Ключевые особенности дизайна

### 1. Типографика
- **Заголовки:** 4xl-5xl для h1, жирные (font-bold)
- **Подзаголовки:** 2xl-3xl для h2
- **Основной текст:** base-lg
- **Вторичный текст:** sm-xs, пониженная непрозрачность

### 2. Интервалы
- **Padding карточек:** lg (24px)
- **Gap между элементами:** 3-4 (12-16px)
- **Margin секций:** 6 (24px)

### 3. Анимации
- **Длительность:** 150-200ms (быстрые и отзывчивые)
- **Easing:** ease-in-out
- **Эффекты:** fade-in, scale минимальные

### 4. Тени
- **Карточки:** shadow-sm (очень легкие)
- **Модальные окна:** shadow-md
- **FAB кнопка:** shadow-lg

### 5. Скругления
- **Кнопки:** rounded-lg (8px)
- **Карточки:** rounded-2xl (16px)
- **Инпуты:** rounded-2xl (16px)
- **Навигация:** rounded-t-2xl (только сверху)

## 🌓 Темная тема
