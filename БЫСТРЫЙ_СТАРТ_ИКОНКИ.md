# ⚡ БЫСТРЫЙ СТАРТ - Добавить иконки за 5 минут

## 🎯 План действий

1. Открой сайт → 2. Скачай набор → 3. Распакуй в папку → Готово! ✅

---

## ⏱️ Вариант 1: САМЫЙ БЫСТРЫЙ (5 минут)

### Шаг 1: Открой Noun Project
🔗 https://thenounproject.com/browse/icons/term/muscle/

### Шаг 2: Скачай нужные иконки
1. Найди иконки мышц (trapezius, biceps, deltoid и т.д.)
2. Кликни на иконку → справа "Download"
3. Выбери **Free** (PNG) или **Pro** (SVG за $3.99/мес)
4. Скачай файлы

### Шаг 3: Конвертируй PNG в SVG (если нужно)
🔗 https://convertio.co/png-svg/ (бесплатный онлайн конвертер)

### Шаг 4: Положи файлы
1. Скопируй SVG файлы в `gym-tracker/public/icons/muscles/[категория]/`
2. Готово! ✅

## ✅ Проверка


## ❓ Проблемы?

### Иконка не показывается?
1. Проверь имя файла (должно совпадать точно!)
2. Проверь путь (папка `back/`, `chest/` и т.д.)
3. Перезагрузи страницу (F5)

### Белый квадрат вместо иконки?
Скачай SVG формат, не PNG!
