# 🔗 АНАТОМИЧЕСКИЕ ИКОНКИ ДЛЯ КОНКРЕТНЫХ МЫШЦ

## ⚠️ ВАЖНО! Что ты ищешь:

Тебе нужны НЕ просто "иконка руки" или "человек", а **АНАТОМИЧЕСКИЕ СХЕМЫ** где выделена конкретная мышца (трапеция, широчайшая, дельты и т.д.), как в учебниках по анатомии.

**ПРОБЛЕМА:** Таких бесплатных готовых паков практически НЕТ!

## 💡 РЕАЛЬНЫЕ РЕШЕНИЯ (Что используют фитнес-приложения)

### ✅ ВАРИАНТ 1: Силуэт тела с зонами (РЕКОМЕНДУЮ!)

Большинство фитнес-приложений используют **простой силуэт тела** с **цветными зонами** для разных мышечных групп.

**Как это выглядит:**
```
     👤
   /  |  \     ← Каждая зона своим цветом
 /    |    \   
  |   |   |    
  |   |   |
```

**Готовое решение в коде:**

```tsx
// src/components/BodyMap.tsx

const MuscleMap = ({ muscle }: { muscle: string }) => {
  const muscleColors: Record<string, string> = {
    chest: '#EF4444',      // Красный
    back: '#3B82F6',       // Синий  
    shoulders: '#F59E0B',  // Оранжевый
    biceps: '#10B981',     // Зелёный
    triceps: '#06B6D4',    // Голубой
    legs: '#8B5CF6',       // Фиолетовый
    abs: '#F97316',        // Оранжево-красный
  };

  return (
    <svg width="200" height="400" viewBox="0 0 200 400">
      {/* Голова */}
      <circle cx="100" cy="50" r="30" fill="#E5E7EB" />
      
      {/* Плечи */}
      <rect 
        x="50" y="80" width="100" height="40" 
        fill={muscle === 'shoulders' ? muscleColors.shoulders : '#E5E7EB'}
        opacity={muscle === 'shoulders' ? 1 : 0.3}
      />
      
      {/* Грудь */}
      <rect 
        x="70" y="120" width="60" height="50" 
        fill={muscle === 'chest' ? muscleColors.chest : '#E5E7EB'}
        opacity={muscle === 'chest' ? 1 : 0.3}
      />
      
      {/* Пресс */}
      <rect 
        x="75" y="170" width="50" height="60" 
        fill={muscle === 'abs' ? muscleColors.abs : '#E5E7EB'}
        opacity={muscle === 'abs' ? 1 : 0.3}
      />
      
      {/* Бицепсы */}
      <ellipse 
        cx="40" cy="150" rx="15" ry="40" 
        fill={muscle === 'biceps' ? muscleColors.biceps : '#E5E7EB'}
        opacity={muscle === 'biceps' ? 1 : 0.3}
      />
      <ellipse 
        cx="160" cy="150" rx="15" ry="40" 
        fill={muscle === 'biceps' ? muscleColors.biceps : '#E5E7EB'}
        opacity={muscle === 'biceps' ? 1 : 0.3}
      />
      
      {/* Ноги */}
      <rect 
        x="75" y="230" width="20" height="150" 
        fill={muscle === 'legs' ? muscleColors.legs : '#E5E7EB'}
        opacity={muscle === 'legs' ? 1 : 0.3}
      />
      <rect 
        x="105" y="230" width="20" height="150" 
        fill={muscle === 'legs' ? muscleColors.legs : '#E5E7EB'}
        opacity={muscle === 'legs' ? 1 : 0.3}
      />
    </svg>
  );
};
```

### ✅ ВАРИАНТ 2: Цветные бейджи + текст (ПРОЩЕ ВСЕГО!)

Самое простое - использовать **цветную метку** рядом с названием мышцы:

```tsx
// src/components/MuscleLabel.tsx

const MUSCLE_COLORS = {
  chest: 'bg-red-500',
  back: 'bg-blue-500',
  shoulders: 'bg-orange-500',
  biceps: 'bg-green-500',
  triceps: 'bg-cyan-500',
  legs: 'bg-purple-500',
  quads: 'bg-indigo-500',
  hamstrings: 'bg-violet-500',
  abs: 'bg-amber-500',
  glutes: 'bg-pink-500',
} as const;

export const MuscleLabel = ({ 
  muscle, 
  name 
}: { 
  muscle: keyof typeof MUSCLE_COLORS; 
  name: string 
}) => (
  <div className="flex items-center gap-2">
    <div className={`w-4 h-4 rounded-full ${MUSCLE_COLORS[muscle]}`} />
    <span className="text-sm font-medium">{name}</span>
  </div>
);

// Использование:
<MuscleLabel muscle="chest" name="Грудь" />
<MuscleLabel muscle="biceps" name="Бицепс" />
```

**2. Создай маппинг для групп мышц:**

```typescript
// src/constants/muscleIcons.ts

  // Используем иконки из Health Icons
  chest: '/icons/outline/body/heart.svg',

  workout: '/icons/outline/exercise/exercise.svg',
  weights: '/icons/outline/exercise/activity-weights.svg',
} as const;
```

**3. Создай компонент:**

```tsx
// src/components/MuscleIcon.tsx

interface MuscleIconProps {
  muscle: keyof typeof MUSCLE_ICONS;
  size?: number;
  className?: string;
}

export const MuscleIcon = ({ muscle, size = 24, className = '' }: MuscleIconProps) => {
  return (
    <img 
      src={MUSCLE_ICONS[muscle]}

      height={size}
      className={className}
    />
  );
};
```


## 🚀 ПОШАГОВАЯ ИНСТРУКЦИЯ (Health Icons)

### Шаг 1: Скачай пак

**Вариант A - Через браузер:**
1. Открой https://healthicons.org/
2. Нажми "Download all" внизу страницы
3. Скачается `icons.zip`

**Вариант B - Через терминал:**
```powershell
cd public
curl -O https://healthicons.org/icons.zip
# Или используй браузер если curl не работает
```

### Шаг 2: Распакуй архив

```powershell
# Распакуй в public/icons/health/
Expand-Archive -Path icons.zip -DestinationPath public/icons/health
```

### Шаг 3: Изучи структуру

После распаковки будет:
```
public/icons/health/
  ├── filled/          # Заполненные иконки
  │   ├── body/
  │   ├── exercise/
  │   └── ...
  └── outline/         # Контурные иконки
      ├── body/
      ├── exercise/
      └── ...
```

### Шаг 4: Создай маппинг

Создай `src/constants/muscleIcons.ts`:

```typescript
export const MUSCLE_ICONS = {
  // ГРУППЫ МЫШЦ

  back: '/icons/health/outline/body/body.svg',
  shoulders: '/icons/health/outline/body/arm.svg',

  // ТИПЫ ТРЕНИРОВОК

  strength: '/icons/health/outline/exercise/activity-weights.svg',
  running: '/icons/health/outline/exercise/activity-run.svg',

export type MuscleGroup = keyof typeof MUSCLE_ICONS;
```

### Шаг 5: Создай компонент

`src/components/MuscleIcon.tsx`:

```tsx

  );

```

```tsx
import React from 'react';

type MuscleGroup = 
  | 'chest' | 'back' | 'shoulders' | 'arms' | 'biceps' | 'triceps'
  | 'legs' | 'quads' | 'hamstrings' | 'calves' | 'glutes'
  | 'abs' | 'core' | 'cardio' | 'fullbody';

  muscle: MuscleGroup;
  size?: 'sm' | 'md' | 'lg' | 'xl';
}

const muscleEmojis: Record<MuscleGroup, string> = {
  chest: '🫀',
  back: '🔙',
  shoulders: '🏔️',
  arms: '💪',
  biceps: '💪',
  triceps: '🦾',
  legs: '🦵',
  quads: '⚡',
  hamstrings: '🦴',
  calves: '🥾',
  glutes: '🍑',
  abs: '⬜',
  core: '🎯',
  cardio: '❤️',
  fullbody: '🏋️',
};

  const sizes = {
    sm: 'text-xl',

      {muscleEmojis[muscle]}
    </span>
  );
};
```

```tsx
  shoulders: '🏔️',
  biceps: '💪',
  triceps: '🦾',
  legs: '🦵',
  abs: '⬜',
  cardio: '❤️',
  strength: '🏋️',
  running: '🏃',
};

interface MuscleIconProps {
  muscle: MuscleGroup;
  size?: number;
  useEmoji?: boolean; // Флаг для переключения на эмодзи
  className?: string;
}

export const MuscleIcon = ({ 
  muscle, 
  size = 24,
  useEmoji = false,
  className = '' 
}: MuscleIconProps) => {
  if (useEmoji) {
    return (
      <span 
        className={`inline-block ${className}`}
        style={{ fontSize: size }}
        role="img" 
        aria-label={muscle}
      >
        {EMOJI_FALLBACK[muscle]}
      </span>
    );
  }

  return (
    <img 
      src={MUSCLE_ICONS[muscle]} 
      alt={muscle}
      width={size}
      height={size}
      className={className}
      onError={(e) => {
        // Fallback на эмодзи если SVG не загрузился
        const target = e.target as HTMLImageElement;
        target.style.display = 'none';
        target.insertAdjacentHTML('afterend', 
          `<span style="font-size: ${size}px">${EMOJI_FALLBACK[muscle]}</span>`
        );
      }}
    />
  );
};
```

**Использование:**
```tsx
// SVG иконка
<MuscleIcon muscle="biceps" size={32} />

// Эмодзи (если хочешь протестировать)
<MuscleIcon muscle="biceps" size={32} useEmoji={true} />
```
