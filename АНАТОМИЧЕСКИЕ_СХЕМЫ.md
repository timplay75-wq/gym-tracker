# 🔗 АНАТОМИЧЕСКИЕ СХЕМЫ МЫШЦ (БЕСПЛАТНО!)

## ⚠️ ЧТО ТЫ ИЩЕШЬ

Тебе нужны **АНАТОМИЧЕСКИЕ СХЕМЫ** отдельных мышц (трапеция, широчайшая, дельты), как в учебниках, а НЕ просто "иконка руки".

**РЕАЛЬНОСТЬ:** Бесплатных готовых паков с такими схемами практически НЕТ.

### 💾 Как скачать и использовать:

1. **Открой ссылку** на нужную мышцу
2. Справа нажми **"More details"**
3. Проверь **лицензию** (обычно Public Domain или CC BY-SA)
4. Нажми **"Download"** → выбери размер (рекомендую 1000px+)
5. **Обработай в Figma/Photopea**:
   - Убери фон (сделай прозрачным)
   - Измени цвет на нужный
   - Экспортируй как SVG или PNG

## 💡 АЛЬТЕРНАТИВНЫЕ РЕШЕНИЯ

### Вариант 1: Цветные бейджи (ПРОЩЕ ВСЕГО!)

Не использовать картинки вообще, а покозать цветную метку:

```tsx
// src/components/MuscleLabel.tsx

const MUSCLE_COLORS = {
  chest: 'bg-red-500',
  back: 'bg-blue-500',
  shoulders: 'bg-orange-500',
  biceps: 'bg-green-500',
  triceps: 'bg-cyan-500',
  legs: 'bg-purple-500',
  quads: 'bg-indigo-500',
  hamstrings: 'bg-violet-500',
  abs: 'bg-amber-500',
  glutes: 'bg-pink-500',
} as const;

export const MuscleLabel = ({ muscle, name }: { muscle: keyof typeof MUSCLE_COLORS; name: string }) => (
  <div className="flex items-center gap-2">
    <div className={`w-4 h-4 rounded-full ${MUSCLE_COLORS[muscle]}`} />
    <span>{name}</span>
  </div>
);

// Использование:
<MuscleLabel muscle="chest" name="Грудь" />
<MuscleLabel muscle="biceps" name="Бицепс" />
```

### Вариант 2: Силуэт тела с SVG

Создать простой силуэт где каждая зона подсвечивается:

```tsx
// src/components/BodyMap.tsx

export const BodyMap = ({ muscle }: { muscle: string  }) => {
  const getColor = (m: string) => muscle === m ? '#3B82F6' : '#E5E7EB';
  const getOpacity = (m: string) => muscle === m ? 1 : 0.3;

  return (
    <svg width="200" height="400" viewBox="0 0 200 400">
      {/* Голова */}
      <circle cx="100" cy="50" r="30" fill="#E5E7EB" />
      
      {/* Плечи */}
      <rect x="50" y="80" width="100" height="40" 
        fill={getColor('shoulders')} opacity={getOpacity('shoulders')} />
      
      {/* Грудь */}
      <rect x="70" y="120" width="60" height="50" 
        fill={getColor('chest')} opacity={getOpacity('chest')} />
      
      {/* Бицепсы */}
      <ellipse cx="40" cy="150" rx="15" ry="40" 
        fill={getColor('biceps')} opacity={getOpacity('biceps')} />
      <ellipse cx="160" cy="150" rx="15" ry="40" 
        fill={getColor('biceps')} opacity={getOpacity('biceps')} />
      
      {/* Пресс */}
      <rect x="75" y="170" width="50" height="60" 
        fill={getColor('abs')} opacity={getOpacity('abs')} />
      
      {/* Ноги */}
      <rect x="75" y="230" width="20" height="150" 
        fill={getColor('legs')} opacity={getOpacity('legs')} />
      <rect x="105" y="230" width="20" height="150" 
        fill={getColor('legs')} opacity={getOpacity('legs')} />
    </svg>
  );
};

// Использование:
<BodyMap muscle="biceps" />
```

### Вариант 3: Создать в Figma (ПРОФЕССИОНАЛЬНО!)

1. Открой **Figma** (бесплатно: https://figma.com)
2. Найди референс анатомии в Google Images
3. Обведи контуры мышц инструментом **Pen**
4. Раскрась каждую мышцу отдельным цветом
5. Экспортируй как **SVG**

**Время:** 2-4 часа для всех мышечных групп

**Плюсы:**
- ✅ Полный контроль над дизайном
- ✅ Точно под твой стиль
- ✅ Можешь продавать/использовать без ограничений

## 💾 ГОТОВЫЙ КОД - Компонент с Wikimedia

```typescript
// src/constants/muscleImages.ts

export const MUSCLE_IMAGES = {
  // Путь к скачанным файлам из Wikimedia
  trapezius: '/images/muscles/trapezius.png',
  lats: '/images/muscles/latissimus-dorsi.png',
  chest: '/images/muscles/pectoralis-major.png',
  deltoids: '/images/muscles/deltoid.png',
  biceps: '/images/muscles/biceps.png',
  triceps: '/images/muscles/triceps.png',
  quads: '/images/muscles/quadriceps.png',
  hamstrings: '/images/muscles/hamstrings.png',
  glutes: '/images/muscles/gluteus-maximus.png',
  abs: '/images/muscles/rectus-abdominis.png',
} as const;

export type MuscleType = keyof typeof MUSCLE_IMAGES;
```

```tsx
// src/components/MuscleImage.tsx

import { MUSCLE_IMAGES, type MuscleType } from '@/constants/muscleImages';

interface Props {
  muscle: MuscleType;
  size?: number;
  className?: string;
}

export const MuscleImage = ({ muscle, size = 100, className = '' }: Props) => {
  return (
    <img
      src={MUSCLE_IMAGES[muscle]}
      alt={muscle}
      width={size}
      height={size}
      className={`object-contain ${className}`}
    />
  );
};

// Использование:
<MuscleImage muscle="biceps" size={150} />
<MuscleImage muscle="chest" size={120} className="opacity-80" />
```
