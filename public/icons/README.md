# Примечание

Иконки для PWA (icon-192x192.png и icon-512x512.png) должны быть добавлены в папку public/icons/.

Вы можете сгенерировать их с помощью онлайн-сервисов, таких как:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Временно можно использовать любые изображения PNG нужных размеров.
