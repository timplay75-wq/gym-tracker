# 🎨 Design Update Summary

## ✅ Что сделано

### 1. Обновлена цветовая схема Tailwind
- Темно-серый акцент (`#202328`)
- Чистый белый фон
- Минималистичная палитра

- **Button**: плоский стиль, минимальный hover

- **Home**: большие заголовки, простая статистика
- **ActiveWorkout**: минималистичный таймер и формы

### 4. Добавлена адаптивность
- Mobile First подход
- Responsive для планшетов и desktop

### 5. Улучшена темная тема
- Плавные переходы
- Кастомные цвета для темной темы
- Автоопределение системной темы


## 🚀 Как использовать

1. **Запустить dev сервер:**
   ```bash
   npm run dev
   ```

2. **Открыть в браузере:**
   ```
   http://localhost:5173/
   ```

3. **Тестировать на разных устройствах:**
   - Desktop: обычное окно браузера
   - Tablet: DevTools → Responsive (768px)
   - Mobile: DevTools → iPhone 12 Pro (390px)

4. **Переключать темы:**
   - Кнопка в правом верхнем углу главной страницы
   - Переключатель в профиле

## 💡 Советы

### Для дальнейшей разработки:
1. Всегда используйте компоненты из `@/components`
2. Следуйте паттерну больших заголовков (text-4xl)
3. Используйте минималистичные тени (shadow-sm)
4. Скругления: кнопки - lg, карточки - 2xl
5. Адаптивность: mobile first (базовые стили → sm → md → lg)

### Цвета по назначению:
```javascript
// Текст
text-text-light-primary dark:text-white     // Основной
text-text-light-secondary dark:text-gray-400 // Второстепенный
text-text-light-tertiary dark:text-gray-500  // Placeholder

// Фон
bg-background-light dark:bg-gray-900        // Страницы
bg-white dark:bg-gray-800                   // Карточки

// Акцент
bg-primary-500                               // Кнопки, активные элементы
```
